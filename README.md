# Nubian Landing Page

This is the landing page for Nubian, a language learning app for African languages.
It features a highly interactive, 3D-enhanced scrolling experience.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: GSAP (ScrollTrigger), Framer Motion
- **3D**: React Three Fiber, Drei
- **Smooth Scroll**: Lenis
